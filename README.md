# Green Destinations - Attrition Analysis

## Project Objective
The HR Director of Green Destinations want to understand why employees are leaving (attrition). This project analyzes the data to find trends related to Age, Income, and Years at the Company.

## Tools Used
* Python
* Pandas (for data calculation)
* Seaborn & Matplotlib (for graphs)

## Key Findings
* **Attrition Rate:** 16.12%
* **Age:** Younger employees are more likely to leave.
* **Tenure:** Employees with 0-2 years at the company have the highest attrition.
* **Income:** Lower income is linked to higher attrition.
